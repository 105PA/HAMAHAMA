# 🦝 HAMA - HAMA 🦛

## 📌 개요
### 📝 서비스 설명
- 스터디 대화 STT 및 화자분리하여 Text로 추출
- 추출한 텍스트를 바탕으로 요약 생성
- 다양한 AI 기술을 활용하여 스터디 공부 예습/복습 지원
